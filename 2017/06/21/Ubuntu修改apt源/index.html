Ubuntu修改apt源

本地安装了一个Ubuntu系统，为了方便使用，把apt源替换为国内的阿里云的源地址

直接设置中去修改

打开SystemSettings，选择Software & Updates 选择中国的镜像就可以。

手动修改配置文件

手动修改配置文件并更新

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

修改配置文件

sudo vim /etc/apt/sources.list
